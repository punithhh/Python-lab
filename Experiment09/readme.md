## Aim

To write a program using Pandas to create and display a DataFrame.

## Algorithm

1. Import `pandas` as `pd`.
2. Create multiple lists such as `names`, `age`, `marks`, `grades`, etc.
3. Create a DataFrame using `pd.DataFrame()` with these columns.
4. Store all data in tabular form.
5. Print the DataFrame using `print()`.
